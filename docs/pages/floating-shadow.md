# Floating Shadow

<p class="uk-text-lead">Create a soft floating shadow below elements to make them appear to hover.</p>

## Usage

To apply a floating shadow at the bottom of an element, add the `.uk-floating-shadow` class. It can be combined with one of the other `.uk-box-shadow-*` modifiers.

```html
<div class="uk-floating-shadow"></div>
```

```example
<div class="uk-floating-shadow uk-box-shadow-small uk-width-1-2@s uk-text-center">
    <div class="uk-background-default uk-padding-large">
        Floating Shadow
    </div>
</div>
```
