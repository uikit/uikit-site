# Floating Shadow

<p class="uk-text-lead">Easily create nice looking floating shadow.</p>

***

## Usage

To apply a floating shadow at the bottom of an element so that appears to be hovering, add the `.uk-floating-shadow` class. This can also be combined with one of the other `.uk-box-shadow-*` modifiers.

```html
<div class="uk-floating-shadow"></div>
```

```example
<div class="uk-floating-shadow uk-box-shadow-small uk-width-1-2@s uk-text-center">
    <div class="uk-background-default uk-padding-large">
        Box shadow bottom
    </div>
</div>
```
